====================================================================
    Empyrion CSV Translation Tool
    バージョン 1.0
====================================================================

【概要】
Empyrion - Galactic Survival のローカライゼーションCSVファイルを
日本語化するためのツールです。

ゲーム本体のバージョンアップに伴い、ローカライゼーションファイルの
項目が増えていきます。このツールは、既に翻訳済みの項目を過去の
ファイルからマージし、新規項目の翻訳作業を効率化します。


【主な機能】
- 既存翻訳データの自動マージ
- 未翻訳部分の自動翻訳（Google翻訳使用）


【使い方】

1. フォルダ構成
   ------------------------------------------------
   Empyrion CSV Tool.exe
   ├─ 1_Original/      # 現行バージョンのCSVファイルを配置
   ├─ 2_Translated/    # 前バージョンなどの翻訳済みファイルを配置
   ├─ 3_Result/        # 出力先（自動作成）
   └─ 4_Error/         # 翻訳エラー行の出力先（自動作成）
   ------------------------------------------------

2. CSVファイルの準備

   【1_Original フォルダ】
   現行バージョンのゲームファイルから以下をコピー：
   - Content\Configuration\Dialogues.csv
   - Content\Extras\Localization.csv
   - Content\Extras\PDA\PDA.csv

   【2_Translated フォルダ】
   前バージョンなど、既に翻訳済みのCSVファイルを配置
   ※同名のファイルがあれば自動的にマージされます
   ※なければ新規翻訳のみ行われます

3. ツールの実行

   (1) Empyrion CSV Tool.exe をダブルクリックして起動
   
   (2) 自動翻訳の設定
       ☑ 自動翻訳する → 未翻訳部分を機械翻訳で埋める
       ☐ 自動翻訳する → マージのみ行い、未翻訳部分は空のまま
       
       ※機械翻訳は補助的なものです。
         精度の高い翻訳が必要な場合は、チェックを外して
         マージのみ行い、後から手動で翻訳することを推奨します。
   
   (3) 「実行」ボタンをクリック
   
   (4) 処理が完了すると、3_Result フォルダに
       完成したCSVファイルが出力されます

4. 翻訳エラーの確認

   翻訳に失敗した行がある場合、4_Error フォルダに
   エラー行のみを抽出したファイルが保存されます。
   これらの行は手動で翻訳してください。


【CSVファイル形式】
処理対象のCSVファイルには以下の列が必要です：
- KEY      : 各行を識別するキー
- English  : 英語の原文
- Japanese : 日本語訳（空の場合に翻訳・マージされます）


【処理の流れ】
1. 1_Original の各CSVファイルを読み込み
2. Japanese列が空の行について、2_Translatedから
   同じKEYの翻訳データをコピー
3. 2_Translatedにもない場合、自動翻訳を実行
   （チェックボックスがONの場合のみ）
4. 完成したファイルを 3_Result に保存
5. エラーがあった行を 4_Error に保存


【注意事項】
- Windows Defender SmartScreenの警告が表示された場合は
  「詳細情報」→「実行」をクリックしてください
  
- 自動翻訳（googletrans）は非公式APIのため、
  大量に使用すると一時的に制限がかかる場合があります
  
- 機械翻訳の精度は完璧ではありません
  重要な部分は必ず手動で確認・修正してください


【タグの扱い】
ゲーム内で使用される特殊タグは自動的に保持されます：
- <color=#ffffff>text</color>
- [b]text[/b]
- [c][ff0000]text[-][/c]
- {variable} （プレースホルダー変数）

これらのタグ構造は保持され、内部のテキストのみが翻訳されます。


【動作環境】
- Windows 10/11 (64bit)
- Pythonのインストールは不要
